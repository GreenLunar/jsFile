About

JsFile is JavaScript library the helps you to work with files in the browser. It works online, without any server-side dependencies. You may open the files (see supported formats), read them (convert the files to HTML), work with file content and styles as with JS objects. Now only "read" mode is available.

Engines

See the available engines that help you to work with different document formats:

  • jsFile-ooxml (.docx, etc.)
  • jsFile-wcbff (.doc, etc.)
  • jsFile-odf (.odf, etc.)
  • jsFile-epub (.epub)
  • jsFile-image (images)
  • jsFile-rtf (.rtf)
  • jsFile-dsv (.csv, .tsv, etc.)
  • jsFile-fb (.fb2)
  • jsFile-txt (simple text files)
